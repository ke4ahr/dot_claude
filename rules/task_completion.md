# Task Completion

ALL steps of a plan, task, or job must be completed.

Do not stop after a subset of steps and report success. If a plan has N steps, execute ALL N steps. If a task has sub-tasks, complete all sub-tasks. Do not skip steps because earlier steps passed or because the work looks "mostly done."

If a step cannot be completed, report it explicitly as incomplete -- do not omit it from the summary.
